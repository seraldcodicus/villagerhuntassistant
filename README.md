# Villager Hunt Assistant - Installation Guide (for now)

## Quick Install (6 steps)

### 1. Download the Extension
- Download **VillagerHunt.zip** from: https://sery.codes/villagerhuntassistant/villagerhunt.test.zip
- Save it to your **Downloads** folder (or wherever you prefer, you can delete it after installation)

> **üìå Note:** Do NOT unzip the file! Chrome will handle this automatically.

### 2. Open Chrome
- Launch Google Chrome browser.

### 3. Navigate to Extensions
- Click the **three dots** (‚ãÆ) in the top-right corner
- Go to **Extensions** ‚Üí **Manage Extensions**
- Or paste this in your address bar: `chrome://extensions`

### 4. Enable Developer Mode
- Find the **Developer mode** toggle in the top-right corner
- Turn it **ON**

### 5. Install the Extension
- Locate **VillagerHunt.zip** in your Downloads folder (or wherever you saved it)
- **Drag and drop** the zip file into the Extensions window
- Chrome will automatically extract and install it

### 6. You're Done!
- Navigate to any Twitch channel's **Mod View** (e.g., `twitch.tv/moderator/risshella`)
- Look for the **island icon** üèùÔ∏è in the top toolbar
- If there's an active hunt, the widget will appear automatically!
- Delete the zip file from your Downloads folder (or wherever you saved it) if you want to. You don't need it anymore.

Do the exact same thing above, just with the new zip file. See why you deleted the old one? So you don't get mixed up! üòâ
